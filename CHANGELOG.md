# Changelog

## 19.01.26
### Fixes
- Исправлен крэш training: `UnboundLocalError: idOfE` в `warhamEnv.py/step()` — удалён дублирующий блок `elif action["attack"] == 0`, который использовал `idOfE` до присваивания.
- Исправлены ошибки отчёта melee-логгера:
  - корректный подсчёт урона из `dmg_list` (numpy/tensor-safe),
  - `invul <= 0` теперь трактуется как “инвула нет” (не “0+ сейв”).

### 10th Edition: Charge & Fight
- Charge phase приведён ближе к 10-й редакции: **чардж не наносит урон**, только фиксирует engagement/факт чарджа.
- Fight Phase включена после хода игрока, чтобы melee реально резолвился и печатались логи.
- В Fight Phase ручные кубы спрашиваются **только за игрока**, а модель кидает автоматически (как в фазе стрельбы).

### UX / Debug
- После успешного charge выполняется обновление доски (`updateBoard()`), чтобы `board.txt` отражал реальную ситуацию “влипли/не влипли”.

## 18.01.26
### Gameplay / Data
- Добавлена фракция Necrons (включая Royal Warden, Canoptek Scarab Swarms).
- Добавлены оружейные абилки: **Lethal Hits**, **Rapid Fire**.
- Добавлено melee-логирование (подробный отчёт по хитам/вунам/сейвам).
