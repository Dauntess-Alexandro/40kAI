# ImGui минимальный GUI для 40kAI

Этот каталог содержит минимальный пример Dear ImGui, чтобы быстро увидеть окно и затем переносить панели из старого GTKmm GUI.

## Что делает пример
- Открывает окно GLFW.
- Рисует простую панель "Командный пункт".
- Рисует панель "Настройки" (перенос первой вкладки из GTKmm).
- Показывает кнопку и счётчик кликов.
- Поддерживает демо-окно ImGui.

## Сборка (Linux/macOS)
```bash
mkdir -p build
cd build
cmake ..
cmake --build .
```

## Сборка (Windows, PowerShell)
```powershell
mkdir build
cd build
cmake ..
cmake --build . --config Release
```

## Что делать дальше
1. Создай следующую панель в `ui_panels.cpp` и перенеси ещё одну секцию из GTKmm.
2. Вынеси следующую часть логики в отдельный файл по примеру `app_state.*`.
3. Постепенно замени старые окна/диалоги на ImGui-панели.

## Примечания
- Проект использует FetchContent и скачивает ImGui + GLFW при конфигурации.
- Если нужен более современный рендер (OpenGL3/DirectX), можно заменить backend на `imgui_impl_opengl3.cpp`.
