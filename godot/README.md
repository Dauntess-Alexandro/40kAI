# 40kAI Viewer (Godot)

Минимальный Godot-проект для просмотра состояния матча из `gui/state.json`.

## Запуск

1. Откройте папку `godot` в Godot 4.x.
2. Нажмите **Play** — откроется сцена `Main.tscn`.

По умолчанию Viewer ищет файл состояния по пути:
`../gui/state.json` (относительно папки `godot`).

Если файл лежит в другом месте — укажите переменную окружения:

```bash
STATE_JSON_PATH=/абсолютный/путь/к/state.json godot4 --path godot
```

## Запуск play.py из Godot

В панели справа есть блок **«Запуск игры»**:

1. Укажите путь к модели (`.pickle`) или оставьте `None`.
2. При необходимости включите опцию **«Запускать с GUI (play.py)»**.
3. Нажмите **Старт** — Godot запустит `play.py` через Python.

Требования:

- установлен Python (по умолчанию используется команда `python`);
- все зависимости проекта установлены (`pip install -r requirements.txt`);
- при необходимости можно задать путь к Python через переменную окружения `PYTHON`.

## Что отображается

- поле боя с сеткой;
- юниты и цели (objectives);
- текущая фаза, активная сторона, VP/CP;
- хвост логов из `response.txt`.
