Задача макетов

Показать, как подсказку можно превратить из справки в быстрый инструмент выбора модели. Я оставил стиль близким к текущему темному интерфейсу, но добавил более ясную группировку, рейтинги и вердикты.

Как читать

A и D лучше для вкладок по семействам. B лучше для сравнения. C лучше для новичка. F самый плотный вариант для опытного пользователя.

DQN

Хороший первый агент для обучения, тестов и сравнения.

+ Сильные стороны
  • понятная логика выбора хода;
  • хороший baseline для новых миссий;
  • быстрее тяжелых search-моделей;
  • удобно сравнивать с PPO и AZ.
! Ограничения
  • может хуже играть сложные тактические позиции;
  • чувствителен к настройкам;
  • требует аккуратного подбора гиперпараметров.
Когда выбирать

Первый запуск, быстрый baseline, проверка train/eval пайплайна.

DQN выбран как стартовая рекомендация

Для короткого train-прогона и проверки пайплайна лучше начать с DQN. Он дает понятный baseline и не требует тяжелого поиска.

Рекомендуется сейчас следом PPO для качества AZ Tree
1 Что сделать

Запустить DQN на текущих настройках, затем сравнить с PPO на том же бюджете эпизодов.

2 Когда менять

Если baseline стабилен, но не хватает тактики, перейти к AZ Tree или GMZ.

Критерий DQN PPO AZ Tree
Быстрая проверка лучше хорошо плохо
Сложная тактика средне средне лучше
Простота объяснения хорошо средне сложнее
