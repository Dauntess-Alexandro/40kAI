AlphaZero Tree

Policy + Value с настоящим MCTS. Сильная тактика, дорогой инференс.

AlphaZero

Кратко

Сеть предлагает ходы, дерево поиска уточняет выбор перед действием.

Лучше всего

Сложная тактика, когда важен просчет вперед.

Осторожно

Может быть слишком дорогим для короткого теста.

Сильные стороны

  • видит продолжения лучше baseline;
  • понятный search-подход;
  • кандидат на качество.

Ограничения

  • дороже по времени;
  • нужен контроль rollout;
  • на LAN важна глубина.

Вердикт

Брать после DQN/PPO, когда baseline уже есть и хочется проверить силу поиска.

Плюс: самый спокойный и понятный вариант для внедрения.
