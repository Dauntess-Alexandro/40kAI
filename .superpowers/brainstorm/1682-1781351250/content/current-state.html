Какая главная цель редизайна?

A

«Состояние модели одним взглядом»

Убрать дубли, наверх вынести 4–5 ключевых чисел + 1–2 главных графика. Остальное — свернуть/ниже.

B

«Площадка для анализа»

Главное — графики. Сделать их крупнее, добавить управление (сглаживание, выбор метрик, зум), шапку ужать в одну строку.

C

«Сравнение моделей»

Дать возможность держать на экране 2 модели рядом / overlay их кривых. Сейчас вкладка строго про одну модель.

D

«Просто чище»

Раскладка по сути ок — нужна косметика: убрать дубли, выровнять, единый стиль, без новой функциональности.
